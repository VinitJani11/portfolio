# 🌐 Vinit Jani Portfolio

A modern, responsive personal portfolio website showcasing my skills, projects, experience, and professional journey as a developer.

## 🚀 Live Demo

**Website:** https://vinitjani11.github.io/portfolio/

## 📋 Overview

This portfolio website serves as a central hub for my professional presence, highlighting:

* Professional introduction and background
* Technical skills and expertise
* Featured projects and achievements
* Contact information and social links
* Responsive user experience across devices

The website is designed to provide recruiters, collaborators, and potential clients with a clear overview of my work and capabilities.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/VinitJani11/portfolio.git
```

### Navigate to the Project Directory

```bash
cd portfolio
```

### Run Locally

Simply open `index.html` in your browser or use a local development server:

```bash
# VS Code Live Server
Right Click → Open with Live Server
```
